# CoE Starter Kit Release Resources

This directory contains resources and documentation for creating and managing CoE Starter Kit releases.

## Contents

- **[RELEASE_PROCESS.md](RELEASE_PROCESS.md)** - Complete release process documentation including critical post-release steps
- **Collateral/** - Release collateral files (readme, icons, etc.)
- **Notes/** - Release notes for each version
- **Pipelines/** - Azure DevOps pipeline definitions for creating releases
- **PowerPlatformHub/** - Power Platform Hub deployment scripts

## Quick Links

- [Release Process Documentation](RELEASE_PROCESS.md)
- [GitHub Releases](https://github.com/microsoft/coe-starter-kit/releases)
- [CoE Starter Kit Download Link](https://aka.ms/CoeStarterKitDownload)

## Important Reminders for Release Managers

⚠️ **CRITICAL**: After publishing a GitHub release, the `aka.ms/CoeStarterKitDownload` link MUST be manually updated to point to the new release assets. This step is not automated.

See [RELEASE_PROCESS.md](RELEASE_PROCESS.md) for complete details.
