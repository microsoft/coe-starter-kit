# CoE Starter Kit - Frequently Asked Questions (FAQs)

This directory contains comprehensive FAQ documents addressing common questions about the Center of Excellence (CoE) Starter Kit.

## Available FAQs

### [FAQ: Power Platform Administrator Role Requirements](./FAQ-AdminRoleRequirements.md)

Addresses questions about admin role requirements for the CoE Starter Kit, including:

- Why Power Platform Admin role is required
- What happens if the admin role is removed after installation
- Whether Environment Admin role can be used instead
- Service principal vs. user account considerations
- Security best practices for admin accounts
- Alternative approaches and their limitations

**Relevant for**: IT administrators, security teams, anyone responsible for CoE Kit installation and maintenance

---

### [FAQ: Inactivity Notifications for Governance](./FAQ-InactivityNotifications.md)

Comprehensive guide to understanding and configuring inactivity notification flows in the Governance/Audit Components, including:

- Which environments are scanned for inactive apps and flows
- How to configure the `admin_ProductionEnvironment` variable
- Changing approval recipients for testing
- Excluding specific environments or resources from notifications
- Managing orphaned apps and flows
- Customizing notification emails
- Testing and troubleshooting
- Best practices for governance rollout

**Relevant for**: CoE administrators, governance teams, anyone deploying or managing inactivity notifications

## Support

The CoE Starter Kit is provided as sample implementations and is not officially supported by Microsoft Support. For issues and questions:

- **Questions about the kit**: Use the [Question issue template](https://github.com/microsoft/coe-starter-kit/issues/new/choose)
- **Bug reports**: Use the appropriate [issue template](https://github.com/microsoft/coe-starter-kit/issues/new/choose)
- **Community support**: Visit the [Power Apps Community forum](https://powerusers.microsoft.com/t5/Power-Apps-Governance-and/bd-p/Admin_PowerApps)
- **Office Hours**: Join [CoE Starter Kit Office Hours](https://aka.ms/coeofficehours)
