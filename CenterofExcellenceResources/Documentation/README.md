# CoE Starter Kit Documentation

This folder contains supplementary documentation for the CoE Starter Kit beyond what's available in the [official Microsoft documentation](https://learn.microsoft.com/en-us/power-platform/guidance/coe/starter-kit).

## Available Documentation

### [TROUBLESHOOTING.md](TROUBLESHOOTING.md)
Quick reference guide for common issues and their solutions. Start here if you're encountering a specific problem during setup or upgrade.

**Use this when:**
- You're stuck on a specific step
- You're seeing an error or unexpected behavior
- You need a quick solution

### [FAQ-Upgrade.md](FAQ-Upgrade.md)
Frequently asked questions about upgrading the CoE Starter Kit, including detailed explanations and background information.

**Use this when:**
- You want to understand why something works the way it does
- You need detailed step-by-step instructions
- You're preparing for an upgrade and want to understand the process

## Contributing

If you've encountered an issue that's not covered in these documents:

1. Check if there's an [existing issue](https://github.com/microsoft/coe-starter-kit/issues) describing the problem
2. If your issue is unique, please [create a new issue](https://github.com/microsoft/coe-starter-kit/issues/new/choose)
3. Consider contributing to this documentation by submitting a pull request
