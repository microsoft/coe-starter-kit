# Center of Excellence - Audit Components

This folder contains the Audit Components solution for the CoE Starter Kit, which includes governance and compliance features for Power Platform.

## Documentation

- **[Inactivity Notifications Configuration Guide](./INACTIVITY-NOTIFICATIONS-CONFIGURATION.md)** - Learn how to configure the 3 weeks, 4 weeks, and 6 months timeframes for inactivity notification approvals

## Key Features

The Audit Components solution includes:

- **Inactivity Notifications** - Automatically detect and manage unused apps and flows
- **Approval Workflows** - Request approval for archiving or deleting inactive resources
- **Compliance Tracking** - Monitor and audit resource usage across your tenant
