# CoE Starter Kit - Additional Documentation

This directory contains supplementary documentation for the Microsoft Power Platform Center of Excellence (CoE) Starter Kit.

## Available Guides

### [Authentication Troubleshooting Guide](Authentication-Troubleshooting.md)

A comprehensive troubleshooting guide for authentication issues with CoE Starter Kit app registrations.

**Covers:**
- Common authentication failures and solutions
- Secret rotation best practices
- Azure Key Vault access issues
- API permission problems
- Diagnostic checklists and scripts

**Best for:** Resolving authentication errors and maintaining healthy app registrations.

### [Workload Identity Federation - Quick Reference](WIF-Quick-Reference.md)

A quick reference guide with immediate answers about Workload Identity Federation impact on CoE Starter Kit.

**Quick Answers:**
- Current status and recommendations
- What to do now vs. later
- Secret management best practices
- Top 5 FAQs

**Best for:** Quick consultation and immediate guidance.

### [Workload Identity Federation Migration Guide](WorkloadIdentityFederationMigration.md)

A comprehensive guide for organizations planning to migrate from client secret-based authentication to Workload Identity Federation (WIF) for their CoE Starter Kit app registrations.

**Topics Covered:**
- Understanding Workload Identity Federation
- Impact assessment on CoE components
- Current Power Platform WIF support status
- Migration approaches and recommendations
- Best practices for secret management
- Detailed FAQ and troubleshooting

**Best for:** Detailed planning and in-depth understanding.

**Who Should Read These:**
- Power Platform Administrators
- CoE Kit Administrators
- Azure Active Directory Administrators
- Security and Compliance Teams
