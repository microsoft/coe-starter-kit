# CoE Starter Kit Troubleshooting Guides

This directory contains troubleshooting guides for common issues encountered when using the Microsoft Power Platform Center of Excellence (CoE) Starter Kit.

## Available Guides

### Core Components

## Getting Help

If you don't find a solution in these guides:

1. **Search existing issues**: Check the [GitHub issues](https://github.com/microsoft/coe-starter-kit/issues) to see if your problem has been reported
2. **Check the wiki**: Visit the [CoE Starter Kit Wiki](https://github.com/microsoft/coe-starter-kit/wiki/Issue-triage-(CoE-Starter-Kit)) for additional resources
3. **Official documentation**: Review the [Microsoft Learn documentation](https://learn.microsoft.com/power-platform/guidance/coe/starter-kit)
4. **File a new issue**: If your problem is new, file an issue using the appropriate [issue template](https://github.com/microsoft/coe-starter-kit/issues/new/choose)
