# CoE Starter Kit - Troubleshooting Guide

This directory contains troubleshooting guides for common issues encountered when setting up and using the CoE Starter Kit.

## Common Issues

### Setup and Installation

- [AppForbidden Error with CoE Setup and Upgrade Wizard](AppForbidden-Setup-Wizard.md) - Resolves permission errors when accessing the Setup Wizard
- [AppForbidden Error - Decision Tree](AppForbidden-Decision-Tree.md) - Quick diagnostic flowchart for permission issues

## Getting Help

If you don't find a solution to your issue in these troubleshooting guides:

1. Check the [official CoE Starter Kit documentation](https://learn.microsoft.com/en-us/power-platform/guidance/coe/starter-kit)
2. Search for [existing GitHub issues](https://github.com/microsoft/coe-starter-kit/issues)
3. If your issue is not already reported, [create a new issue](https://github.com/microsoft/coe-starter-kit/issues/new/choose)

## Contributing

Found a solution to a common issue? Consider contributing to this troubleshooting guide by:

1. Creating a new markdown file in this directory following the same format
2. Updating this README with a link to your troubleshooting guide
3. Submitting a pull request

For more information on contributing, see the [How to Contribute](../../HOW_TO_CONTRIBUTE.md) guide.
