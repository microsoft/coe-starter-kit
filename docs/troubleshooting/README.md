# CoE Starter Kit Troubleshooting Guides

This directory contains troubleshooting guides and issue response templates for common CoE Starter Kit problems.

## Available Guides

### [Dataflow Refresh Not Working](./dataflow-refresh-not-working.md)
Issue response template for when CoE dataflows are not refreshing sequentially or automatically.

**Use this when:**
- Only the Maker dataflow refreshes
- Subsequent dataflows show old timestamps
- Dataflow chain is not working as expected

## Using These Guides

### For Users
If you're experiencing an issue with the CoE Starter Kit:

1. Browse the troubleshooting guides in this directory
2. Follow the step-by-step resolution instructions
3. If the issue persists, open a GitHub issue with the information requested in the guide

### For Maintainers
When responding to issues:

1. Check if there's a matching troubleshooting guide
2. Use the response template as a starting point
3. Customize based on the specific user's situation
4. Link to the full troubleshooting documentation for detailed steps

## Contributing

If you've encountered and resolved a common issue:

1. Create a new troubleshooting guide following the existing format
2. Include:
   - Clear symptom description
   - Root cause explanation
   - Step-by-step resolution
   - Validation steps
   - Related resources
3. Submit a pull request
