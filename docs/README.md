# CoE Starter Kit Documentation

This directory contains troubleshooting guides and additional documentation for the Microsoft Power Platform Center of Excellence (CoE) Starter Kit.

## Troubleshooting Guides

### Active Sync Flow Errors

If you're experiencing errors with sync flows in the CoE Admin Command Center:

- **[Quick Fix Guide](QUICK-FIX-SYNC-ERRORS.md)** - Start here for fast solutions to common issues
- **[Detailed Troubleshooting](TROUBLESHOOTING-SYNC-FLOW-ERRORS.md)** - Comprehensive step-by-step troubleshooting

### Common Issues Covered

- Failed to escalate privileges errors
- Connection authentication problems
- Permission issues
- API throttling and limits
- Flow configuration after upgrades
