# CoE Starter Kit Documentation

This directory contains troubleshooting guides and documentation for the Microsoft Power Platform Center of Excellence (CoE) Starter Kit.

  - Other Core Component apps

- **[FAQ - Common Issues](FAQ-COMMON-ISSUES.md)** - Quick reference for frequently asked questions and common problems with the CoE Starter Kit

## Getting Help

If you need help with the CoE Starter Kit:

1. **Check the troubleshooting guides** in this directory
2. **Search existing issues** in the [GitHub Issues](https://github.com/microsoft/coe-starter-kit/issues)
3. **Review the wiki** at [Issue Triage](https://github.com/microsoft/coe-starter-kit/wiki/Issue-triage-(CoE-Starter-Kit))
4. **File a new issue** using the [Bug Report Template](https://github.com/microsoft/coe-starter-kit/issues/new/choose)
5. **Ask the community** in the [Power Apps Community Forum](https://powerusers.microsoft.com/t5/Power-Apps-Governance-and/bd-p/Admin_PowerApps)
