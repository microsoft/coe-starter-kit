## Getting Help

If you encounter issues with the CoE Starter Kit:

1. **Check existing documentation**
   - Review the [official documentation](https://docs.microsoft.com/power-platform/guidance/coe/starter-kit)
   - Check the troubleshooting guides in this directory

2. **Search for existing issues**
   - Browse [GitHub Issues](https://github.com/microsoft/coe-starter-kit/issues)
   - Look for closed issues that may have addressed your problem

3. **Submit a new issue**
   - If your problem is not already documented, submit a new issue at [aka.ms/coe-starter-kit-issues](https://aka.ms/coe-starter-kit-issues)
   - Use the appropriate issue template for bugs, features, or questions

4. **Community support**
   - Join discussions in the [Power Apps Community](https://powerusers.microsoft.com/t5/Power-Apps-Governance-and/bd-p/Admin_PowerApps)

## Disclaimer

The CoE Starter Kit represents sample implementations of Power Platform features. While the underlying platform features are fully supported, the kit itself is provided as-is. For issues with:
- **The kit**: Use [GitHub Issues](https://github.com/microsoft/coe-starter-kit/issues)
- **Power Platform features**: Contact Microsoft Support through your standard support channel
